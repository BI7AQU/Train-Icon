# 列车图标模型
##### 本项目受[CrYinLang/EmuAIO](https://github.com/CrYinLang/EmuAIO)启发，使用群友贡献的图标包结合人工智能绘制而成。

##### 文件夹结构：

- train：原始图片

- asset：使用AI生成的图片

- icon：人工修改后的图标


##### 能力有限，如有错误欢迎提交issues。也欢迎大佬提交PR。
